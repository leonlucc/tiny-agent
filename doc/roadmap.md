# 渐进式演进路线图 (Full Roadmap)

Tiny Agent 坚信“最好的学习方式是亲手构建”。为了避免一上来就面对复杂的完整系统，本项目将 Agent 的核心能力拆解到不同的 Git Tag 中。

你可以通过以下命令切换到任意版本进行独立运行和源码研读：
```bash
git checkout v0.3.0  # 示例：切换到 RAG 阶段
```
---

### [v0.0.1] Hello LLM

* **定位**：项目起步，跑通最基础的闭环。
* **核心能力**：学习如何配置和初始化大语言模型（LLM）SDK，实现单轮文本输入与输出，建立流式异常处理机制。
* **UI演进**：本版本无前端 UI，纯 CLI 终端交互，方便开发者聚焦后端网络请求本身。

### [v0.1.0] Chat

* **定位**：从“一问一答”走向“连续对话”。
* **核心能力**：设计后端 Chat History（上下文记忆）的数据结构，引入 SSE (Server-Sent Events) 协议实现打字机式的流式响应。
* **UI演进**：引入极简单页（输入框 + 消息列表 + 发送按钮），用原生 JS 动态解析并渲染流式文本。

### [v0.2.0] Prompt Engineering

* **定位**：让模型听话，按规矩办事。
* **核心能力**：引入 System Prompt 机制锁死 Agent 角色，实现结构化输出（Structured Outputs），强制 LLM 稳定返回 JSON 格式数据。
* **UI演进**：侧边栏增加配置面板，支持动态修改 System Prompt、Temperature 等模型参数。

### [v0.3.0] RAG

* **定位**：给 Agent 灌入外挂知识库，拒绝胡说八道。
* **核心能力**：对接 Embedding 模型实现文本向量化，基于纯 Python 或轻量本地向量库打通“检索 -> 拼装 Prompt -> 生成”的 RAG 全链路。
* **UI演进**：增加知识库管理模块，支持前端拖拽上传文档。

### [v0.4.0] Tool Calling

* **定位**：赋予 Agent 改变世界、连接外部系统的双手。
* **核心能力**：深入 Function Calling 原理，学习后端声明工具、解析 LLM 调用意图并在本地执行函数，适配现代开源标准 MCP (Model Context Protocol)。
* **UI演进**：在对话流中高亮展示 Tool 的调用状态与链路（参数 → 调用状态 → 返回结果）。

### [v0.5.0] Agent Loop

* **定位**：从被动触发到自主思考（Agent 的智能核心）。
* **核心能力**：手写 ReAct 范式核心循环，让 Agent 学会拆解复杂任务，自主进行 Thought -> Action -> Observation 的状态循环直至得出答案。
* **UI演进**：动态可视化 Agent 思考过程，前端展示思考状态卡片，让用户看清 Agent 每一步的执行逻辑。

### [v1.0.0] Tiny Agent MVP

* **定位**：核心里程碑，标准 Agent 体系大成。
* **核心能力**：将 LLM + Prompt + Tool + RAG + Loop 深度整合，形成高内聚、低耦合的单体 Agent 架构，优化并发性能与长对话内存清理。
* **UI演进**：打通所有模块，交付全功能的 Agent 综合控制面板，兼具调试与日常使用属性。

### [v1.1.0] Skills

* **定位**：生产级能力进阶，探索工具池的大规模扩展。
* **核心能力**：引入 Skills 概念，实现工具的动态注册、发现、按需选择以及多工具的自动串联编排。
* **UI演进**：提供 Skills 管理界面，支持一键启用/禁用特定技能，监控技能调用频次。

### [v1.2.0] Memory

* **定位**：生产级能力进阶，突破 Token 窗口限制。
* **核心能力**：设计记忆漏斗模型，实现短期记忆、中期记忆（语义检索摘要）与长期记忆（键值对/图谱），支持定时后台异步压缩摘要。
* **UI演进**：提供 Memory 管理页面，允许开发者直接查看、检索、甚至手动编辑 Agent 的深层记忆碎片。

### [v1.3.0] [Multi-Agent]

* **定位**：生产级能力进阶，走向多智能体协作。
* **核心能力**：定义不同角色的子 Agent，手写任务分发与状态路由机，实现智能体之间的静态和动态协作/监督机制。
* **UI演进**：提供专用的 Agent 属性面板，方便查看和配置每个 Agent 的角色和当前状态。
